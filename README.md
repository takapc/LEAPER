# LEAP 英単語クイズ

LEAP改訂版の英単語をランダムに出題して答えを確認できるWebアプリケーションです。

## 機能

- ランダムに出題: 指定した範囲の英単語をランダムに表示
- 答えの表示/非表示: ボタンで答えを確認できます
- 次の問題: ランダムに次の単語を表示
- オフラインデータ: JSONファイルから単語データを読み込み
- データのインポート: JSONファイルやHTMLから単語を手動でインポート可能

## セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

## 単語データについて

### デフォルトのデータ

アプリは `src/data/words.json` から単語データを読み込みます。このファイルにはサンプルデータが含まれています。

### 完全なデータの追加方法

LEAP改訂版の全2300語を追加するには、以下のいずれかの方法を使用できます：

#### 方法1: JSONファイルを直接編集

`src/data/words.json` を編集して、以下の形式で単語データを追加してください：

```json
[
  {
    "id": 1,
    "word": "agree",
    "meaning": "[自] ①賛成する ②（主語の中で）意見が一致する ③（with ～）（気候，食べ物が）（～に）合う"
  },
  {
    "id": 2,
    "word": "oppose",
    "meaning": "[他] ～に反対する"
  }
]
```

#### 方法2: アプリ内でインポート

1. アプリの「データをインポート」ボタンをクリック
2. 「JSONファイル」タブを選択
3. JSONファイルを選択するか、JSON形式のテキストを貼り付け
4. 「JSONをインポート」をクリック

データはブラウザのローカルストレージに保存され、次回起動時も使用できます。

#### 方法3: HTMLからインポート

1. [LEAP 改訂版 単語一覧](https://ukaru-eigo.com/leap-modified-list/) のページを開く
2. ページのHTMLをコピー（右クリック → ページのソースを表示、または開発者ツールでHTMLをコピー）
3. アプリの「データをインポート」ボタンをクリック
4. 「HTML/テキスト」タブを選択
5. コピーしたHTMLを貼り付けて「HTMLをインポート」をクリック
